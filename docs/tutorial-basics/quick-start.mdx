# Quick Start

Here are full example that you can just copy and use it on your project

### SDK Installation

<Tabs>
  <TabItem value="js" label="JS/TS">
    ```bash
    npm i mugshot-js
    npm i mugshot-js/@types
    ```
  </TabItem>
  <TabItem value="go" label="Go">
    ```bash
    go get github.com/mugshot-dev/mugshot-go@latest
    ```
  </TabItem>
  <TabItem value="python" label="Python">
    ```bash
    pip3 install mugshot
    ```
  </TabItem>
</Tabs>

Please reach us if you need SDK implementation in another language
